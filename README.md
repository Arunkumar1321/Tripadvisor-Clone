# 🌍 TripAdvisor Clone

This is a responsive front-end clone of the TripAdvisor website built using HTML and CSS. The project focuses on modern layout techniques, animations, and responsive design principles.

## 🚀 Features

* Responsive design using media queries
* Flexbox-based layout for clean structure
* Smooth hover and transition animations
* Navigation bar with interactive elements
* Image cards for destinations, hotels, and activities
* Mobile-friendly UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Animations & Transitions
* Media Queries

## 📱 Responsiveness

The website is fully responsive and adapts to:

* Desktop screens
* Tablets
* Mobile devices

## 🎯 Purpose of the Project

This project was built to practice:

* Layout design using Flexbox
* Responsive web design
* UI structuring similar to real-world websites
* CSS animations and hover effects

## 📸 Sections Included

* Header / Navigation bar
* Search section
* Categories (Things to do, Hotels, Restaurants)
* Featured destinations
* Footer

## ⚙️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Run the `index.html` file in any web browser
